AMX Mod X plugin for Counter-Strike.

Adds a surfboard model.

## Cvars
- ```surf_board_spawn_give "1"``` Give boards to all players on respawn.
- ```surf_board_back "1"``` Display the board model on the player's back.
- ```surf_board_player_animation "1"``` Play player's idle animation while surfing.
