_[English](README.md) | **Русский**_

![Surf Board](https://github.com/user-attachments/assets/d0e3a86b-6931-4fda-ae55-8a87667a5b18)

AMX Mod X плагин для Counter-Strike.

Добавляет модель доски для серфинга.

## Квары
- ```surf_board_spawn_give "1"``` Выдавать всем игрокам при возрождении.
- ```surf_board_back "1"``` Отображать модель на спине игрока.
- ```surf_board_player_animation "1"``` Проигрывать idle анимацию игрока во время серфинга.
